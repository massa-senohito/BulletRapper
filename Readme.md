Bulletのラッパーライブラリです。

・Bulletの
BulletCollision_Debug.lib
bulletcollision_debug.pdb
BulletDynamics_Debug.lib
bulletdynamics_debug.pdb
BulletRapper.lib

このプロジェクトはBulletをVisualStudio2010でビルドしたため、
他の依存しているプロジェクトもVisualStudio2010プロフィールでビルドしています。
